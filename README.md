# Bio-Update-Tracker

## 基本功能

- 自动检测玩家 Bio 是否发生变化
- 在聊天中提示有更新的角色
- 点击“查看变化”可打开对比界面
- 高亮显示新增和删除内容（逐行 + 行内差异）
- 支持深色/浅色切换与快捷关闭

## 安装方法

### 前置要求
1. 安装浏览器脚本管理器：
   - Tampermonkey（油猴）或 Violentmonkey（篡改猴）
2. 安装并启用 WCE（BC 扩展）
3. 在 WCE 设置中开启：
   - “保存并浏览已知的个人资料（需要刷新）”

### 安装
- Tampermonkey / Violentmonkey
   - https://raw.githubusercontent.com/thisaueser/Bio-Update-Tracker/main/main.user.js
